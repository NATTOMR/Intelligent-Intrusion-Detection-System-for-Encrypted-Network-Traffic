Generalization Test — UNSW-NB15
Cross-dataset evaluation on 246,996 completely unseen network flows.

Dataset Shift Analysis

Our Hybrid IDS was trained entirely on CICIDS2017 traffic. Testing on UNSW-NB15 — a dataset with different feature names, attack families, and lab environments — revealed significant performance degradation.

Why this matters: This is a classic example of Dataset Shift (Covariate Shift). Only ~24 of 78 required CICIDS2017 features could be semantically mapped from UNSW-NB15. This finding motivates the need for transfer learning, domain adaptation, and online/continual learning in production IDS deployments.
